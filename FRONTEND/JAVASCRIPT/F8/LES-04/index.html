JavaScript Basic

Chuỗi trong JavaScript
1. Cách tạo chuỗi
- Có 3 cách để tạo chuỗi trong JavaScript:

  • Single quotes
  • Double quotes
  • Backticks
Ví dụ:
  • let single = 'single-quoted';
  • let double = "double-quoted";
  • let backticks = `backticks`;
2. Một số case sử dụng backslashes
- Đôi khi, chúng ta cần sử dụng backslashes để thêm ký tự đặc biệt vào chuỗi.
Ví dụ:
  • let str = "Hello\nWorld"; // includes a newline
  • let str = "Hello\\World"; // includes a backslash
  • let str = "Hello\tWorld"; // includes a tab
3. Xem độ dài chuỗi
- Để xem độ dài của chuỗi, chúng ta sử dụng thuộc tính length.
Ví dụ:
  • let str = "Hello";
  • alert( str.length ); chuỗi có độ dài là 5
4. Template string ES6
- Template string là một chuỗi đặc biệt cho phép chèn biến và biểu thức vào bên trong.
- Template string bắt đầu bằng dấu nháy huyền (backticks) và kết thúc bằng dấu nháy huyền.
- Để chèn biến vào chuỗi, chúng ta sử dụng ${...}.
Ví dụ:
  • let name = "John";
  • let str = `Hello, ${name}!`;
  • alert(str); // Hello, John!
