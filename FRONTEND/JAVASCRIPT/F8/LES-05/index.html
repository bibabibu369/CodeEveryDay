Javascript Basic

Lam viec voi chuoi

- Làm việc với chuỗi trong Javascript:

  1. Tạo chuỗi
  2. Nối chuỗi
  3. Tìm kiếm chuỗi
  4. Cắt chuỗi
  5. Thay thế chuỗi
  6. Chuyển đổi chuỗi
  7. So sánh chuỗi
  8. Lấy độ dài chuỗi
  9. Làm việc với chuỗi và mảng
- Tạo chuỗi:
  • Đơn giản: var str = "Hello World";
  • Đa dòng: var str = "Hello\nWorld";
  • Đặc biệt: var str = "Hello\"World";
- Nối chuỗi:
  • var str1 = "Hello";
  • var str2 = "World";
  • var str3 = str1 + " " + str2;
- Tìm kiếm chuỗi:
  • var str = "Hello World";
  • var pos = str.indexOf("World");
  • var pos = str.lastIndexOf("World");
- Cắt chuỗi:
  • var str = "Hello World";
  • var res = str.slice(1, 5);
  • var res = str.substring(1, 5);
  • var res = str.substr(1, 5);
- Thay thế chuỗi:
  • var str = "Hello World";
  • var res = str.replace("World", "Vietnam");
- Chuyển đổi chuỗi:
  • var str = "Hello World";
  • var res = str.toUpperCase();
  • var res = str.toLowerCase();
- So sánh chuỗi:
  • var str1 = "Hello World";
  • var str2 = "Hello World";
  • var res = str1.localeCompare(str2);
- Lấy độ dài chuỗi:
  • var str = "Hello World";
  • var len = str.length;
- Làm việc với chuỗi và mảng:
  • var str = "Hello World";
  • var arr = str.split(" ");
